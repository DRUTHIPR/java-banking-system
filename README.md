# Java Banking System

A console-based banking management system developed using Java and Object-Oriented Programming concepts.

## Features

- Customer management
- Bank account creation
- Deposit money
- Withdraw money
- Transfer money between accounts
- Account search functionality
- Transaction history
- Menu-driven banking interface


## Technologies Used

- Java
- IntelliJ IDEA
- Object-Oriented Programming
- ArrayList Collections


## OOP Concepts Implemented

- Classes and Objects
- Constructors
- Encapsulation
- Object Association
- Methods
- Collections

## How to Run

1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run Main.java
4. Use the menu options to perform banking operations
